# LUXE DRIVE — Premium Car Rental

A modern, minimalist car rental website built with **Next.js 14**, **shadcn/ui**, **Tailwind CSS**, and **Framer Motion**.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- `/` — Homepage with hero, featured fleet, how-it-works, testimonial
- `/fleet` — Full catalog with category filters, search, sort, price range
- `/fleet/[id]` — Car detail page with specs, pricing, booking CTA
- `/booking?id=` — 3-step booking flow with add-ons and order summary
- `/about` — Company story, stats, values, team
- `/contact` — Contact form, info cards, locations

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS** with shadcn/ui design tokens
- **shadcn/ui** components (Button, Card, Badge, Input, Separator)
- **Lucide React** icons
- **Framer Motion** for animations
